# Smart Attendance System

An automated biometric attendance system utilizing deep learning for real-time face recognition.

## 🚀 Features
- Real-time multi-face detection and identification.
- Web-based dashboard for attendance tracking.
- Database integration for persistent records.

## 🧠 Model Details
This system utilizes the **InsightFace** library with the following architecture:
- **Detection:** SCRFD (Sample and Computation Redistribution for Efficient Face Detection).
- **Recognition:** MobileFaceNet.
- **Model Pack:** buffalo_s.

## 🛠️ Tech Stack
- **Language:** Python
- **Web Framework:** Flask
- **Database:** SQLite
- **Computer Vision:** OpenCV, InsightFace
